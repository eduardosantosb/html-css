Estilos Externos

Por que utilizar?

Os estilos externos em CSS são mais versáteis pois eles tornam o código HTML mais limpo e facilitam a estilização de um projeto com duas ou mais páginas.

Para utilizar utilizar o CSS dentro de um arquivo HTML, basta usar a tag <link> stylesheets e criar uma nova pasta sendo style.css no head do HTML.
